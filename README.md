# 블로그 미니 프로젝트

## 개요

게시글, 카테고리, 댓글 기능을 제공하는 블로그 게시판 웹 애플리케이션입니다.
사용자는 브라우저에서 게시글을 작성, 조회, 수정, 삭제할 수 있으며, 게시글에 카테고리를 설정하고 익명 댓글을 작성할 수 있습니다.

## 주요 기능

### 게시글 관리
- 게시글 생성 / 수정 / 조회 / 삭제
- 한개 페이지에 게시글 10개씩 조회
- 조회수 기능 추가
- 공개 / 비공개 기능 추가

### 태그 관리
- 태그 생성 / 수정 / 조회 / 삭제


### 댓글 관리
- 댓글 생성 / 수정 / 조회 / 삭제
- 대댓글 기능 추가
